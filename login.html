To enable Admin Panel on GitHub Pages, you need to configure Firebase Authentication.

Quick Setup (5 minutes):

  1. Create Firebase Project:
    • Go to https://console.firebase.google.com/
    • Click "Add project" and follow the setup wizard
  2. Enable Authentication:
    • In Firebase Console, go to Authentication → Sign-in method
    • Enable "Email/Password" provider
  3. Enable Realtime Database:
    • In Firebase Console, go to Realtime Database
    • Click "Create Database"
    • Select "Start in test mode" (for now)
  4. Get Configuration:
    • Go to Project Settings → General → Your apps
    • Click "" icon to add web app
    • Copy the config object
    • Paste it into js/firebase-config.js
  5. Create Admin User:
    • Run: node scripts/setup-firebase-admin.js
    • Follow the prompts to create admin credentials
    • Import the generated JSON into Firebase Realtime Database
